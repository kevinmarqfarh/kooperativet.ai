# Roadmap

## Phase 0: Prototype

- Forum dashboard and thread detail mockups.
- Prompt bank, build logs, MVP scene, governance and moderation surfaces.
- Language and theme settings.
- Open-source contribution and governance docs.

## Phase 1: Community Foundation

- Supabase Auth and profiles.
- Topics, comments, votes, bookmarks and flags persisted in Postgres.
- Role-based permissions for members, moderators and admins.
- Manual moderation queue with decision logs.
- Code of conduct acceptance during onboarding.

## Phase 2: Knowledge Systems

- Prompt versions, model metadata and eval receipts.
- Build logs that can be followed and converted into guides.
- Review rooms with feedback modes and constructive downvote rules.
- Search across discussions, prompts, users and projects.

## Phase 3: Cooperative Mechanics

- Proposal and voting workflows.
- Steward roles for guilds and knowledge areas.
- MVP matching, sponsorship circles and opt-in inclusion spaces.
- Transparent decision archive.

## Phase 4: Launch Readiness

- Security review and threat model.
- Accessibility audit.
- Abuse testing for spam, brigading and report floods.
- Privacy and data retention review.
- Public beta launch.
